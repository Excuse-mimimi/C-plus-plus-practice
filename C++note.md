#### *数据类型*
##### 2.3 *复合类型*（compoud type）
1.**引用**（reference）
 * **引用**（reference）为对象起了另外一个名字，引用类型引用（refers to）另外一种类型。
 * 通过将声明符写成&d的形式来定义引用类型，其中d是声明的变量名。
  ```c++
    int i = 1024;
    int &d1 = i;        //d1指向i
    int &d2;            //错误，引用必须被初始化
    auto &h = 42;       //错误，不能为非常量引用绑定字面值
    const auto &j =42;  //正确，可以为常量引用绑定字面值
  ```
